# reader

Web reader with translation and furigana annotation support

[Try it now](https://vlzx.github.io/reader/)

## Features

- [ ] Translation
- [ ] Japanese morphological analysis
  - [ ] Furigana(振り仮名) annotation
  - [ ] POS tagging
- [ ] Parsing
  - [ ] Plain text
  - [ ] EPUB
- [ ] Mobile responsive
